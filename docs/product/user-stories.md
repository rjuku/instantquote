# User Stories

- As a plumber, I want to record a short note after a visit so I do not forget job details.
- As an electrician, I want a draft quote with missing prices left blank so I can add accurate costs later.
- As a decorator, I want assumptions and exclusions separated so the customer sees only appropriate notes.
- As a handyman, I want a readable Markdown draft so I can review the quote before using a full web app.
